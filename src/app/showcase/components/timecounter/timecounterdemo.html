Time Counter Time Counter is a component that is useful to Count down any event.

Import


import {TimecounterModule} from 'primeng/timecounter';

Getting Started

Countdown starts when you provide end date.


import {Component} from '@angular/core';

@Component({
    template: '<p-time-counter [endDate]="endDate"></p-time-counter>',
})
export class TimeCounterDemo implements OnInit {
     endDate:string;
     ngOnInit(): void {
        let endingDate = new Date();
        endingDate.setDate(endingDate.getDate() + 7);
        this.endDate = endingDate.toDateString();
    }
    
}

Properties

Name Type Default Description
endDate string null Ending Date and time for counter
defaultStyle boolean true Change to false if you want to apply custom css and use style or styleClass
style string null Inline style of the component.
styleClass string null Style class of the component.

Events

Name Parameter Description
counterEnd event: Browser Event Callback to invoke when the date of counter is less than current date
